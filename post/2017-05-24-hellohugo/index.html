それらのメモ

hugo とは

https://gohugo.io/
  • 静的サイトジェネレーターであり、テンプレートなどを書いてbuildするとHTMLが出力される。
  • 記事などをmarkdownで書ける。
  • HTMLで出力されるのでGithubPagesなどで運用できる。
  • などなど…

Wordpressを運用するほどでもないブログを作成するにはもってこいなので今回はこちらで一からテンプレートを作ってGithubPagesで運用することにした。

インストール

私はwinとmacを使うので両方のインストール方法を書いておく。

まずはGoのインストールをする。
https://golang.org/

$ go version
go version go1.8.2 windows/amd64

go get した時のPATHを通す。

windowsでは

C:\Users\ユーザー名\go\bin

macの場合は

export GOPATH="$HOME/.go"
export PATH="$PATH:$HOME/.go/bin"

を.bush_profileなどに書いておく。

次にhugoのインストール。

$ git clone https://github.com/spf13/hugo.git
$ cd hugo
$ go get

$ hugo version
Hugo Static Site Generator v0.22-DEV windows/amd64 BuildDate: 2017-05-24T10:53:10+09:00

テンプレート作成

プロジェクトファイルを作成する。

$ hugo new site hoge

とりあえず起動する。

$ cd hoge
$ hugo server

多分何も表示されない。
テンプレートは /layouts に作成する。

$ cd layouts
$ echo "<p>hogehoge</p>" > index.html

もう一度起動すると何かしら出るはず。
テンプレートの話はまたいつか書きます。
