hugoのテンプレートの話

hugoでブログを作る際に一からテンプレートを作成した。
それらのメモ。

hugo のインストール

前回の記事を参照。

テンプレート作成

テンプレートは　Project/layouts/以下に作る。
以下はこのサイトのlayouts。

$ tree /f
└─layouts
   │  404.html
   │  index.html
   │
   ├─partials
   │      footer.html
   │      head.html
   │      header.html
   │      pagenav.html
   │
   └─_default
           list.html
           single.html
           terms.html
  • トップページ　→　index.html
  • 404　→　404.html
  • カテゴリ、タグ　→　list.html
  • カテゴリ、タグ一覧　→　terms.html
  • 記事　→　_default/single.html

となっている。
他にもpageとかあるけど必要に応じて作る。
ちなみにGithubPagesなら自動で404.htmlにリダイレクトしてくれるっぽい。

cssとかjsとか画像とか

テンプレートタグ

layouts/partials/に置いたhtmlは以下のように記述すると呼び出すことができる

{{ partial "header.html" . }}

使いまわしたいパーツなどはpartials/以下においてじゃんじゃん使いまわしていく。

また、config.tomlに以下のように書いておいくと

[Params]
hoge = "hoge"
fuga = "fuga"

以下のように呼び出す事ができる。

<p>{{ .Site.Params.hoge }}</p>
<p>{{ .Site.Params.fuga }}</p>
